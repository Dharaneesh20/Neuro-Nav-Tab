## ğŸ¯ About Neuro-Nav Widescreen

Neuro-Nav Widescreen Edition is a modern, responsive web application built with **Flask** and **Tailwind CSS**, specifically optimized for:
- **iPad and iPad Pro** tablets
- **Android Tablets** (Samsung, Google Pixel Tablet, etc.)
- **Desktop Browsers** on widescreen monitors
- **Chromebooks** and other tablet devices

The application helps neurodivergent individuals and people with sensory sensitivities navigate the world with less sensory stress through AI-powered routing, community support, and wellness tools.

### Core Mission
**Making cities more accessible for neurodivergent individuals.** By prioritizing sensory wellness in navigation, we create personalized, stress-free navigation experiences across all platforms.

### Option 2: Local Development Setup

#### Prerequisites
- **Python** 3.9 or higher
- **pip** (Python package manager)
- **Git**
- **Google Cloud Account** (for OAuth - optional for local testing)

#### Installation Steps

1. **Clone Repository**
   ```bash
   git clone https://github.com/Dharaneesh20/Neuro-Nav-Tab.git
   cd Neuro-Nav-Tab
   ```

2. **Create Virtual Environment**
   ```bash
   # On macOS/Linux
   python3 -m venv venv
   source venv/bin/activate

   # On Windows
   python -m venv venv
   venv\Scripts\activate
   ```

3. **Install Dependencies**
   ```bash
   pip install -r requirements.txt
   ```

4. **Set Up Environment Variables**
   ```bash
   cp .env.example .env
   ```

   Edit `.env` with your configuration:
   ```env
   # Flask Configuration
   SECRET_KEY=your-very-secret-key-here-min-32-chars
   FLASK_ENV=development
   FLASK_DEBUG=1

   # Google OAuth (Optional for local development)
   GOOGLE_CLIENT_ID=your-google-client-id.apps.googleusercontent.com
   GOOGLE_CLIENT_SECRET=your-google-client-secret

   # Application Settings
   APP_NAME=Neuro-Nav
   DATABASE_URL=sqlite:///neuro_nav.db
   ```

5. **Run Development Server**
   ```bash
   python app.py
   ```

   The app will be available at: **http://localhost:5000**

6. **Access the Application**
   - Open browser to `http://localhost:5000`
   - Test on tablet by accessing `http://[YOUR_IP]:5000` from tablet

## ğŸ” Google OAuth Setup (Production)

### Step 1: Create Google Cloud Project

1. Visit [Google Cloud Console](https://console.cloud.google.com/)
2. Click "Select a Project" â†’ "NEW PROJECT"
3. Enter project name: `Neuro-Nav`
4. Click "CREATE"

### Step 2: Enable Required APIs

1. Go to "APIs & Services" â†’ "Library"
2. Search for "Google+ API"
3. Click "ENABLE"

### Step 3: Create OAuth Credentials

1. Go to "APIs & Services" â†’ "Credentials"
2. Click "Create Credentials" â†’ "OAuth 2.0 Client ID"
3. Select "Web application"
4. Set Name: `Neuro-Nav Web App`
5. Add Authorized redirect URIs:
   ```
   http://localhost:5000/authorize
   http://localhost:5000/callback
   https://yourdomain.com/authorize
   https://yourdomain.com/callback
   https://neuronavwidescreen.vercel.app/authorize
   https://neuronavwidescreen.vercel.app/callback
   ```
6. Click "CREATE"
7. Copy **Client ID** and **Client Secret**

### Step 4: Configure Application

Add to your `.env`:
```env
GOOGLE_CLIENT_ID=YOUR_CLIENT_ID.apps.googleusercontent.com
GOOGLE_CLIENT_SECRET=YOUR_CLIENT_SECRET
GOOGLE_REDIRECT_URI=https://yourdomain.com/callback
```

## ğŸŒ Deployment Guide

### Deploy to Vercel (Recommended for Web Apps)

Vercel provides free hosting, automatic deployments, and global CDN.

#### Step 1: Prepare Repository

1. Push code to GitHub:
   ```bash

       "GOOGLE_CLIENT_ID": "@GOOGLE_CLIENT_ID",
       "GOOGLE_CLIENT_SECRET": "@GOOGLE_CLIENT_SECRET"
     },
     "functions": {
       "api/**/*.py": {
         "runtime": "python3.9"
       }
     }
   }
   ```

#### Step 2: Deploy on Vercel

1. Visit [vercel.com](https://vercel.com/)
2. Sign up with GitHub account
3. Click "New Project"
4. Select your `Neuro-Nav-Tab` repository
5. Configure project:
   - **Framework Preset**: Other
   - **Build Command**: `pip install -r requirements.txt && python app.py`
   - **Output Directory**: `.`
6. Add environment variables:
   - `SECRET_KEY`
   - `GOOGLE_CLIENT_ID`
   - `GOOGLE_CLIENT_SECRET`
7. Click "Deploy"

**Live URL**: `https://neuronavwidescreen.vercel.app/`

### Deploy to Google Cloud Platform

For more control and scalability:

#### Step 1: Install Google Cloud SDK

```bash
# Follow: https://cloud.google.com/sdk/docs/install

# Initialize
gcloud init
gcloud auth login
gcloud config set project your-project-id
```

#### Step 2: Create App Engine Configuration

Create `app.yaml`:
```yaml
runtime: python39

env: standard
entrypoint: gunicorn -b :$PORT app:app

env_variables:
  FLASK_ENV: "production"
  SECRET_KEY: "your-secret-key"

automatic_scaling:
  min_instances: 1
  max_instances: 10
```

#### Step 3: Deploy

```bash
gcloud app deploy app.yaml --set-env-vars="SECRET_KEY=your-key,GOOGLE_CLIENT_ID=your-id"
```

Your app will be available at: `https://your-project-id.appspot.com/`

## ğŸ› Troubleshooting

### Common Issues

| Issue | Solution |
|-------|----------|
| **Google OAuth fails** | Verify redirect URIs in Google Console match deployment URL |
| **Map not loading** | Check internet connection; ensure APIs are enabled |
| **Slow performance** | Clear browser cache; disable extensions; try incognito |
| **Dark mode not working** | Check browser localStorage is enabled |
| **Offline features unavailable** | Service Worker may not be registered; try hard refresh |

### Development Issues

**Port already in use:**
```bash
# Kill process on port 5000
lsof -ti:5000 | xargs kill -9
# or change port
flask run --port 5001
```

**Module import errors:**
```bash
# Reinstall requirements
pip install --force-reinstall -r requirements.txt
```

**OAuth redirect loop:**
1. Clear browser cookies
2. Check `GOOGLE_REDIRECT_URI` in `.env`
3. Verify Google Cloud Console settings
